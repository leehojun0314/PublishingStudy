span 
: 단순 글자 태그 (아무러 기능 없이 단순히 글을 쓰고 싶을 때)
-----------------------------------------------------------------------------------
div 
: 분할 태그, 상자태그
본래의 태그의 목적은 웹사이트의 영역을 나눠줄 때 사용하는 태그
(영역 관련 얘기는 추후에 자세히 설명)
그 외의 목적은 단순히 사각형, 네모 등을 만들어 줄 때 사용하는 태그
-----------------------------------------------------------------------------------
a 
: #을 써 놓는게 기본 규칙. 링크를 클릭하여 이동시켜줌
-----------------------------------------------------------------------------------
h 
: 제목으로 분류할 수 있는 글자들에 사용
-----------------------------------------------------------------------------------
strong
: 중요한 내용이나 주의사항등을 표기 할 때 사용
-----------------------------------------------------------------------------------
b
: 아무런 역할이 없다. 단순히 굵은 글자를 작성하고 싶을때 사용
-----------------------------------------------------------------------------------
em
: 중요한 내용, 강조하고 싶은 내용 표기할 때 사용
(strong태그랑 역할은 같지만 강조의 방식이 글자를 옆으로 기울여서 강조하는 것)
-----------------------------------------------------------------------------------
i
: 전문용어, 관용어, 기타 기술용어 등등 중요한 학술적 내용을 담고있는 경우 사용
-----------------------------------------------------------------------------------
cite
: 내가 작성한 내용에 대한 출처를 표시할 때 사용
-----------------------------------------------------------------------------------
address
: 연락처 표시할 때 사용
-----------------------------------------------------------------------------------
mark
: 글자 주변에 배경색이 칠해지면서 글자를 강조하고 싶을때 사용
(background-color를 사용하는게 편리하다)
-----------------------------------------------------------------------------------
del
: 기존의 내용을 취소하고 새로 갱신된 내용을 작성하기 위해 사용하는 태그
-----------------------------------------------------------------------------------
ins
: 글자 밑에 밑줄이 생김. 글자를 강조하고 싶을 때 사용가능
(mark, em, strong태그 같은 역할)
-----------------------------------------------------------------------------------
abbr
: 그냥 사용하면 아무 효과 없음. title="원하는 내용"을 작성하면 ...이 생기면서 마우스를 올리면 title에 작성한 내용이 출력되도록 해주는 태그
보통 어떠한 내용의 줄임말, 약자, 약어 등 이 단어가 어떤 내용의 줄임말인지 표기할 때 사용
-----------------------------------------------------------------------------------
sub(아래첨자), sup(위첨자)
: 어떠한 글이나 단어에 기호표시 등 추가적인 내용을 작성할때 사용.
-----------------------------------------------------------------------------------
blockquote
: 길이가 긴 인용문
(인용문: 다른 매체, 인물, 영화, 기타등등 제 3의 무언가가 어딘가에서 사용한 말들을 가져와 쓰는 것)
-----------------------------------------------------------------------------------
small
: 작은 글을 적고싶을 때 사용
