table
: 표를 작성해줄 때 사용하는 태그 (ul 태그 같은 느낌)
tr : 행태그

td : 열 태그

colpsan: 가로 열을 병합해주는 기능으로 입력한 값 만큼 셀 병합이 이루어진다.
주의 사항: 값을 적어줄 때 해당 table태그의 최대 td갯수를 초과해서 적으면 적용되지 않는다.

th : 표의 서브 제목
특징: 작성한 글자가 굵은 글자로 표시되며
글자가 자동으로 태그내에서 가로세로 중앙정렬이 되어있다.

★table태그의 경우 1개의 열에 적용된 가로값은 같은 열에 동시에 적용된다.

rowspan
: 세로 열을 병합해주는 기능으로 입력한 값 만큼 병합이 이루어진다. 다만 colspan과는 다르게
자기 자신도 값에 포함해서 적어줘야 하기 때문에 병합하고 싶은 칸 + 1 해서 작성해줘야 한다.

thead
: 표 안에서 웹사이트의 header영역 처럼 상단 부분을 담당하는 태그내에서(표의 제목이 들어가는 영역)

tbody
: 표 안에서 웹사이트의 main영역 처럼 본문 내용을 담당하는 태그
(tr, td등 표의 핵심내용들을 이루는 대다수의 태그들이 들어가는 영역)

tfoot
: 표 안에서 웹사이트의 footer영역처럼 하단 부 내용을 담당하는 태그
(표의 기타정보, 마무리 내용 등등이 들어가는 영역)
