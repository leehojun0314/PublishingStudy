입력양식태그 교재2권 54페이지~

웹사이트에서 내용을 입력받을 수 있는 모든것들을 만드는 태그

form

ul , table 태그와 비슷한 개념
입력양식태그들을 작성하기 전 기본적으로 작성해주는 태그
(입력양식태그는 내용을 입력받을 수 있는 양식을 만들어주는 태그로 다양한 내용을 입력받을수는 있지만 이 태그를 통해서 어떤 기능등을 만들어주는건 할 수 없다 이 태그들에 기능을 만들어주는 영역은 프론트엔드 분야에서 담당한다 우리는 형태만 만들어주고 끝)

웹디자인 , 웹퍼블리싱 , 프론트엔드 , 백엔드


웹디자인 , 웹퍼블리싱 = 단순히 겉으로 보이는 형태만 만들어주는것

프론트엔드 = 형태만 있는 웹사이트에 실제로 동작을 하는 기능을 넣어주는 역할

백엔드 = 어떤 웹사이트건 회원정보라던지 주문정보 기타 여러가지 정보들을 담아두는 서버라는게 있는데 이 서버를 관리하는 역할

------------------------------------------------------------
입력양식태그의 종류

1.input type text
단순히 아무글자나 내용들을 입력받을 수 있는 공간을 만들어줄 때 사용하는 태그

★2.password
생성된 공간에 글자를 입력하면 특수문자로 표기가 된다
보통 웹사이트에서 비밀번호와 같이 중요한 개인정보를 입력받을 때 사용하는 태그

★★<input type="?" placeholder="원하는내용">
placeholder안에 작성한 내용이 입력양식태그에서 내용을 입력받기 전 평소에 보이는 텍스트로 출력된다

3.search
검색창을 만들어주는 태그로 내용을 입력하면 X표시가 박스 오른쪽에 생기면서 X표시를 눌러주면 입력한 내용이 지워진다

4.url
웹사이트 주소를 입력받는 공간을 만들어줄 때 사용하는 태그

5.tel
pc에서 해당영역을 클릭해서 내용을 작성할 때는 아무런 효과가 없지만 스마트폰에서 해당 영역을 터치하면 다른 input태그들과 다르게 숫자 키패드가 올라온다

6.email
웹사이트에서 이메일 주소 입력받는 공간을 만들어 줄 때 사용하는 태그

7.number
<input type="number" min="1" max="30" step="1" value="1">

min = 최소값 / max = 최대값 / step = 한 번 클릭에 변동되는 값 / value = 사이트를 처음 열었을 때 보이는 값

원하는 값의 범위를 지정해두고 클릭 또는 직접 값을 입력해주는 방식을 통해서
값을 입력받는 공간을 만들어주는 태그

8.range
막대바를 하나 생성해서 이 막대바를 움직여서 number처럼 지정된
범위내에서 값을 입력받을때 사용하는 태그

★9.radio
체크를 할 수 있는 원이 생기며 name과 함께 사용을 하는 태그
원을 체크해서 정보를 입력받는 방식

(웹사이트에서 설문조사 같은 몇 가지 선택지 중 하나를 선택해야하는 것들을 만들어줄때 사용한다)

<input type="radio" name="원하는이름">

name의 경우 이름이 같아야 하나의 그룹으로 인식해서 여러개의 선택지 중에 단 하나만 선택할 수 있게 만들어준다 만약 name에 작성한 내용이 서로 다를경우 다른 그룹으로 인식해서 중복체크가 가능해진다

★checked를 name옆에 작성해주면 사이트를 처음 열었을 때 처음부터 해당 항목에 체크가 된 상태로 출력된다

★10.checkbox
웹사이트에서 어떠한 내용에 대하여 동의를 구하거나 체크가 필요한 상황일 때
사용하는 태그로 radio태그의 경우 여러개의 선택지 중에 하나만 선택하는게 목적인
태그라고 한다면 checkbox는 중복체크가 가능하며 여러개의 선택지가 아닌
한 개의 선택지에 대한 동의 , 체크 , 확인을 구할때 사용하는 태그

11.color
웹사이트에서 색상값을 입력받을 때 사용하는 태그
원하는 색상을 rgb / hsl / hex코드 방식으로 입력받을 수 있다

12.date
연도 / 월 / 일 - 날짜 정보를 입력받는 양식을 만들어주는 태그
(주문제작 상품 , 축하화환 등 정확한 날짜에 배송을 받아야 하는 상품등에 많이 사용한다 , 달력모양 아이콘을 눌러주면 달력으로
날짜 확인)

12-1.week
연도와 그 연도에 몇 번째 주 인지 날짜 정보를 주단위로 입력받을 수 있는 양식을 만들어주는 태그

12-2.datetime-local
date와 똑같은데 + 오전 , 오후 , 시 , 분 등 시간정보까지 같이 입력받을 수 있는 양식태그

★13.file
파일을 업로드 해줄 수 있는 양식을 만들어주는 태그로
실제 파일선택이라는 버튼을 클릭해서 원하는 파일을 업로드 해주는게 가능하다

★업로드한 파일을 다른사람에게 보낸다거나 서버에 올려놓는다던지 하는등의 실제 기능은 프론트엔드 분야의 작업이기 때문에
우리는 양식만 만들어주면 된다

★★label

input태그와 연동해서 사용하는 태그로 input태그의 기능을
그대로 가져와서 대신 수행하는 역할

input태그의 경우 태그의 종류마다 수정할 수 있는 범위가
제한적이다 그래서 input태그를 대신해서 비교적 수정이 자유로운
label태그를 이용해 내가 원하는 형태로 만들어준 다음 사용하기 위함

label태그는 display:inline속성이기 때문에 가로세로 적용이
안되기 때문에 display:block속성으로 바꿔주신 다음 사용하는게 좋다

★label태그와 input태그 연동하는 방법
<label for="연동하고싶은 input태그의 아이디명">
(클래스는 연동X)

14.submit
같은 form태그 내부에 있는 다른 input태그들에 작성한 내용들을
submit 태그를 클릭하면 내가 지정해둔 어딘가로 정보를 제출할 때 사용하는 태그

★button태그 , submit태그와 기능은 동일하다 다만 수정이 가능한 태그

select / option

클릭을 했을때 내가 작성해둔 option의 갯수와 내용에 따라서
선택을 할 수 있는 선택지가 나오는 태그
(쇼핑몰 등에서 다양한 옵션중에 하나를 선택해야하는 양식을 만들어 줄 때 사용하는 태그)

textarea태그
input type text와 비슷한 역할입니다 다만 태그 오른쪽 하단에
마우스 커서를 올려두면 형태가 변하는데 클릭 드래그를 통해
글자를 입력하는 공간의 크기를 조절할 수 있는 태그
