주석 처리

괄호 안에 작성한 내용은 실제 웹사이트에 영향을 주지 않는 코딩화면에서만 보이는 메모기능

웹사이트의 경우 실제로 html, css 파일의 길이가 상당한데 본인이 만든 사이트의 경우 어디서부터 어디까지가 무슨 기능을 담당하고 있고 수정은 어떻게 해야하는지 나는 알 수 있다. 하지만 제 3자 입장에서는 이것들을 알기가 쉽지 않기 때문에 누가 보더라도 쉽게 이해할 수 있도록 주석처리를 통해서 메모를 남겨두는 역할로 사용한다.

ctrl + / 한줄 통채로 주석처리
shift + alt + a 드래그 한 영역만 주석처리
